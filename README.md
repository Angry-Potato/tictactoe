# TDDTic-Tac-Toe
A game of tic-tac-toe played between two murderous machines.

# To install
Checkout the repository using git. Navigate to the root directory, and run:
```sh
npm install
```

# To run tests
From the root directory, run:
```sh
npm test
```
  
# To run the game
From the root directory, run:
```sh
node modules/main.js
```
